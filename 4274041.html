○小夜
果然是最难BOSS，小夜攻击套路很简单，碰到地面必杀不用砥石可以说就直接GG了，躲啊甚么的都是骗人的
个人推荐练到18级，装备上朝岚村正(烈风)、孙六兼元、雾雨村正(丁半斩),这些都是攻击比较高的太刀，准备2~3个伊予砥石（三河有售）
地面时，尽量避免正脸扇的防御，一下半灵力伤不起，多使用闪避饶后打，一开场的灵符不用管，多点用居合斩来解决
小夜怒是有套路的，要么是被烟花炸弹炸到，要么是自己丢3次烟花炸弹，所以在第一次飞天后尽量在第一次丢烟花炸弹的时候把她打下来，千万不要直接把她炸下来， 怒了以后就看运气了。。如果地面必杀了用烈风靠近，等时间过，运气好不出地面必杀注意稳扎稳打，空中必杀毫无威胁除非不小心被大烟花炸弹给爆了_(:3」∠)_

死狂难度一滴血，为了节省时间和避免不必要的战斗，可以搭配青铜镜选择较短路线：

1.山城(黒猿)
2.山城 → 伊賀save(大百足)
3.伊賀back → 伊勢 → 尾張save → 美濃(虎姫)
4.尾張back → 三河 → 遠江(小夜)
5.尾張back → 伊勢 → 伊賀 → 大和(土蜘蛛)
6.尾張back → 美濃 → 飛騨 → 信濃(血狂毘沙門)
7.信濃 → 甲斐 → 駿河save(龍神)
8.駿河back → 伊豆 → 相模 → 武蔵(狗頭龍)

路上也可以捡到足够多的砥石以应付BOSS战斗，不够的话可以路上也可以补给：
大村の砥石×3个、伊予の砥石×1个、五十嵐の砥石×2个、常見時の砥石×3个、内曇り砥石×4个

第二结局要搭配「憑き落とし」「無銘玉ノ緒」穿戴等级要45级，不够的话可以去刷飛騨的雪女魔窟。
第三结局要锻造「朧村正」但穿戴等级需要94级，搭配上经验增加的饰品和刀，刷上一两个小时基本也差不多了。
BOSS战基本都是无脑转圈圈的道具流，熟练转圈圈的时机和及时回复灵力就好了（实在打不过的情况下再无脑道具吧

鬼助篇结局一：打最后boss，吃个鸭锅，用怨霊飛ばし无脑按圈，因为鬼助结局一boss算打俩，砥石带满打，基本上打一两回就过了(打老头不会死，第二个用道具的时候有时会被弄死）。

百姬篇结局一：
吃个自动复活料理打，百姬结局一容易被石头砸死，只能看眼力了，总之方块不松基本上不会死。没啥赖招，只能平砍加眼力了。（百姬结局三吃了个鸭锅，无脑怨霊飛ばし过了，结局一的没有怨霊飛ばし用）

鬼助篇或百姬篇结局2带上打boss不用吃鸭锅，先把刀打断，然后用内曇り砥石，无脑放怨霊飛ばし，打到百姬或者鬼助动弹不了到死。（任何人形带刀能防御的boss，都可用这招打）
