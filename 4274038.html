1. 青坊主
2. save@ 相模 -> 武藏 -> 轮入道
3. 青铜镜返回#相模 -> 伊豆 -> save@ 骏河 -> 雪之丞
4. 青铜镜返回#骏河 -> 甲斐 -> 信浓 -> save@ 飞弹 -> 一本抠脚大汉
5. 青铜镜返回#飞弹 -> save@ 美浓 -> 鵺
6. 青铜镜返回#美浓 -> 近江 -> save@ 山城 -> 大鬼
7. 青铜镜返回#山城 -> save@ 伊贺 -> 伊势 -> 雷神
8. 青铜镜返回#伊贺 -> 大和 -> 不动明王
恭喜跳杯！

说明：一路上遇到小贩优先买青铜镜，6块就够了，其余买烟玉，可以适量买加血的；第6幕山城进入boss区域后，牛头马面遭遇战一律烟玉闪人；注意第2幕中武藏、第7幕中伊势不！要！存！档！；boss房间的记录点可以放心存档，不会覆盖之前青铜镜的返回位置，如果boss战死了，撕游戏读档重置时间；本人在最后boss处记录时用时2小时13分钟，供参考。

贴一个自己的鬼助的路线，可以参考下流程都差不多，不过感觉百姬更快些

1.[序幕]山城(黒猿)
2.[序幕]山城 → 伊賀[保存](大百足)
3.[返回]伊賀 → 伊勢 → 尾張[保存] → 美濃(虎姫)
4.[返回]尾張 → 三河 → 遠江(小夜)
5.[返回]尾張 → 伊勢 → 伊賀 → 大和(土蜘蛛)
6.[返回]尾張 → 美濃 → 飛騨 → 信濃[保存](血狂毘沙門)
7.[返回]信濃 → 甲斐 → 駿河[保存](龍神)
8.[返回]駿河 → 伊豆 → 相模 → 武蔵(狗頭龍)

PS：路上就顺路捡砥石就好了，烟玉可以多买一些，风筝群怪直接跳，其他的道具没必去收集
最花时间的也就是最后BOSS直接在武蔵的行商哪里买内曇り砥石，直接用奥义无脑直接磨光就行了
